> 一个基于 RSS 的信息流推荐工具。支持个性化推荐、多平台同步、自动发现、自定义配置、AI 大模型等功能。

## 🏠 主页

[https://github.com/CaoMeiYouRen/rss-zero#readme](https://github.com/CaoMeiYouRen/rss-zero#readme)


## 📦 依赖要求


- node >=18

## 🚀 安装

```sh
npm install
```

## 🛠️ 开发

```sh
npm run dev
```

## 🔧 编译

```sh
npm run build
```

## 🔍 Lint

```sh
npm run lint
```
